Macros available:
LUNARIS_HEADER_ONLY	                        : Set this if you don't want to use the libs (so it will assume you have the whole project included already)
LUNARIS_AUDIO_SAMPLE_AMOUNT		            : Set a SAMPLE amount (defaults to 8) (has to be a NUMBER!)
LUNARIS_IO_NO_NEW_LINE						: Disables automatic new line after each cout << ... call
LUNARIS_IO_DO_NOT_RESET_COLOR_EACH_LINE		: Disables automatic color reset after each cout << ... call
LUNARIS_ALPHA_TESTING						: Enables alpha stuff that can potentially not work
